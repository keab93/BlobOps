# Devops Project using Docker Compose, CI/CD and Kubernetes

This project uses a simple online game as the playground for implementing and demonstrating DevOps tools and practices. Using Docker Compose, I will spin up bot players of the game and a web server to host the game. The bots will be configured to play against each other, and the web server will provide a user interface to observe the game in real-time.

The project also includes a CI/CD pipeline to automate the building, testing, and deployment of the application. I plan to deploy the application on Kubernetes to demonstrate container orchestration and management in a production-like environment.

## Technologies Used

-Docker Compose: For defining and running multi-container Docker applications.

-CI/CD Tools such as Jenkins to automate the build, test, and deployment processes

-Kubernetes: For container orchestration and management (planned for future implementation).
